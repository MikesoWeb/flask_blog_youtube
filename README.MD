    пользователь может зарегистрироваться на блоге, а так же залогиниться
    конфигурация использует переменные окружения, с помощью модуля python-dotenv

    использованы модули:

        flask flask-wtf
        flask-login
        flask-sqlalchemy
        flask-bcrypt
        python-dotenv
        email-validator
